ARCHITECTURE HIGHLIGHTS
━━━━━━━━━━━━━━━━━━━

6-Level File Precedence:
  1. /etc/scred/config.yaml (system-wide)
  2. ~/.scred/config.yaml (user home)
  3. scred-{ENV}.yaml (environment-specific)
  4. ./scred.yaml (current directory)
  5. $SCRED_CONFIG_FILE (env variable)
  6. Built-in defaults

Per-Path Rules (scred-proxy only):
  - Path-based routing with wildcard support
  - Configurable redaction per path
  - Custom pattern tiers per path
  - Reason/comment fields for documentation

Configuration Sections:
  - scred-cli: Mode, streaming, patterns
  - scred-proxy: Listen, upstream, redaction, per-path rules
  - scred-mitm: Listen, upstream proxy, redaction, CA cert
