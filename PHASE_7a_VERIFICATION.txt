DEPLOYMENT INSTRUCTIONS

1. Release Build:
   cargo build --release
   
   Binaries:
   • target/release/scred-config (library)
   • target/release/scred (CLI tool)
   • target/release/scred-proxy (reverse proxy)
   • target/release/scred-mitm (MITM proxy)

2. Configuration:
   Create scred.yaml in deployment directory:
   
   scred-proxy:
     listen:
       port: 9999
     upstream:
       url: "https://backend.example.com"

3. Systemd Service:
   [Unit]
   Description=SCRED Proxy
   After=network.target

   [Service]
   Type=simple
   ExecStart=/usr/local/bin/scred-proxy
   Restart=on-failure

   [Install]
   WantedBy=multi-user.target

4. Hot-Reload:
   systemctl reload scred-proxy  # Sends SIGHUP
