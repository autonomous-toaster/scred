ROOT CAUSE ANALYSIS
═══════════════════════════════════════════════════════════════════════════════

Root Cause 1: Three Different Redaction Engine Types
  - RedactionEngine: No selector support (used in Proxy/MITM)
  - ConfigurableEngine: Selector support (used only in CLI)
  - StreamingRedactor: No selector support (used in streams)
  Problem: No unified interface, inconsistent implementations

Root Cause 2: Type System Constraints
  - http_proxy_handler expects Arc<RedactionEngine>
  - Can't pass Arc<ConfigurableEngine> without trait
  - Would require Arc<dyn Redactor> trait object
  - Blocker: Refactoring shared code (risk of breaking changes)

Root Cause 3: Code Duplication
  - HTTP proxy logic in 3 places
  - No single place to enforce selectors
  - Each location has different redaction engine
  - Updates to one don't automatically apply to others

Root Cause 4: Missing Integration Tests
  - Test suite doesn't compare CLI vs Proxy vs MITM
  - Doesn't verify selector consistency
  - Doesn't catch inconsistencies across tools
  - P0#1 acknowledged but not fully tested

KEY INSIGHTS & LESSONS LEARNED
═══════════════════════════════════════════════════════════════════════════════

Insight 1: Negative Bias Works
  By assuming code is broken and comparing implementations,
  we found a systematic security gap that positive testing missed.

Insight 2: Integration is the Problem
  Selectors were IMPLEMENTED in CLI and WORKING.
  Problem: Never integrated into production tools.
  Lesson: Implementation done ≠ Integration done

Insight 3: Code Duplication Multiplies Problems
  HTTP proxy logic in 3 places means:
  - Fix must be made 3 times
  - Easy to miss one location
  - Inconsistency results

Insight 4: Silent Failures are Worst
  No error message means:
  - Users don't know it's broken
  - Configuration appears to work
  - Worst type of security bug

Insight 5: Type System Matters
  Type system constraints (Arc<RedactionEngine> not trait)
  prevent using better abstractions.
  Lesson: Design for flexibility early
